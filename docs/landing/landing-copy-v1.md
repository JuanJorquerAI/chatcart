# ChatCart — Landing copy v1

## Hero
**Vende por WhatsApp, conectado a tu WooCommerce.**

Convierte conversaciones en compras: catálogo, carrito, pago y confirmación automática.  
Y para clientes recurrentes: **repetir última compra en segundos**.

CTA primary: **Agendar demo**  
CTA secondary: **Ver demo (WhatsApp)**

Microtrust:
- Integración: WooCommerce + WhatsApp Cloud API
- Pagos: Transbank (Webpay) + opciones configurables

---

## Problema (dolor)
Hoy tus ventas por WhatsApp dependen de humanos, se pierde tiempo, se cometen errores y el cliente se cae en el checkout.

## Solución
ChatCart es un canal de ventas conversacional que:
- identifica al cliente por teléfono
- muestra catálogo/stock/precios desde tu tienda
- arma carrito y crea el pedido
- cobra con el medio que configures
- sincroniza todo de vuelta a WooCommerce

---

## 3 beneficios (bullets)
- **Más conversión:** menos fricción que el checkout web en móvil.
- **Más recompra:** botón de **Repetir última compra** para clientes frecuentes.
- **Menos soporte:** estado del pedido y tracking en el mismo chat.

---

## Cómo funciona (pasos)
1) El cliente escribe por WhatsApp.
2) ChatCart detecta si es nuevo o recurrente.
3) El cliente navega, agrega al carrito y paga.
4) Se crea el pedido en WooCommerce y se envía confirmación.

---

## Para quién es
Tiendas WooCommerce en Chile/LatAm con compras repetibles:
- alimentos
- mascotas
- farmacia
- ferretería
- retail con despacho local

---

## Integraciones
- **WooCommerce** (productos, stock, precios, pedidos)
- **WhatsApp Cloud API** (Meta)
- **Pagos**: Transbank Webpay Plus (MVP) + plugin para otros (Flow/Khipu/MercadoPago/link externo)

---

## Seguridad y control
- Multi-tenant con aislamiento por tienda
- Webhooks firmados (Meta/Woo/pagos)
- Auditoría por conversación
